Welcome to Clonifier

With this program you can:

  • Browse sequence files, e.g. genomes and plasmid maps
  • Simulate PCR
  • Simulate DNA assembly methods such as Gibson Assembly
  • Work in an aesthetically pleasing environment ;)

Because of the questionable design and excessive complexity of this software, you will probably need to start by following the step-by-step Tutorial. You can also read the (probably largely redundant) Guide.

If anything doesn't work properly, it might be because it doesn't work properly, but it might also be your browser. Since the real reason for Clonifier's existence is to play around with new technologies like WebAssembly, it probably only works in the latest versions of Firefox and Chrome.

For more info, go to https://github.com/dlesl/clonifier.
